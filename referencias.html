Tags HTML

Tag Descrição
<blockquote> Porção de texto que cita um conteúdo de fonte externa.
<button> Usado para definir um botão clicavel.
<canvas> Usada para desenhar gráficos, dinamicamente, por meio de scripts.
<cite> Define o título de uma obra criativa.
<del> Define um segmento de texto que foi deletado.
<details> Seção da página que contém informações específicas, e que podem ser ocultadas.
<dfn> Especifica um termo que vai ser definido dentro do conteúdo.
<dt> Define um termo/nome em uma lista de descrição.
<figcaption> Usado para definir uma legenda para uma foto.
<header> Representa um contêiner para conteúdo introdutório (cabeçalho) ou um conjunto de links de navegação.
<hr> Define uma quebra temática em uma página.
<input> Campo de entrada de dados para o usuário.
<ins> Define um texto que foi inserido em um documento.
<label> Usado para definir rótulos para elementos.
<legend> Usado para definir uma legenda para um elemento.
<mark> Define o texto que deve ser marcado ou destacado.
<pre> Exibe um texto pré-formatado.
<progress> Elemento que descreve o progresso de uma tarefa.
<samp> Usada para definir uma amostra do output de um programa de computador.
<select> Elemento que lista opções em um formulário.
<small> Usada para definir um texto de tamanho menor.
<span> Usado para marcar uma parte do texto colorindo essa parte.
<u> Destaca porções de texto que distoam do conteúdo.
<var> Usada para definir uma variável em programação ou em uma expressão matemática.
