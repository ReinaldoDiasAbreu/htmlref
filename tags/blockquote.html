Tag HTML <blockquote>

Exemplo

Uma seção que cita uma outra fonte:

<blockquote cite="https://www.pensador.com/citacoes/">
Creio que o nosso Pai Celeste inventou o homem por estar desapontado com o macaco.
</blockquote>

Definição e uso

A tag <blockquote> especifica uma seção que cita uma outra fonte.

Geralmente os navegadores adicionam indentação ao elemento <blockquote> (veja no exemplo abaixo como remover essa indentação).

Dica: Utilize <q> para citações curtas.

Atributos

Atributo Valor Descrição
cite URL Especifica a fonte da citação

Atributos Globais

A tag <blockquote> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <blockquote> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Utilize CSS para remover a indentação do elemento:

<html>
<head>
<style>
  blockquote {
  margin-left: 0;
}
</style>
</head>
<body>

<blockquote cite="https://www.pensador.com/citacoes/">
Creio que o nosso Pai Celeste inventou o homem por estar desapontado com o macaco.
</blockquote>

</body>
</html>

CSS padrão

Exemplo

blockquote {
  display: block;
  margin-top: 1em;
  margin-bottom: 1em;
  margin-left: 40px;
  margin-right: 40px;
}
