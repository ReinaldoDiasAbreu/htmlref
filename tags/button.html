Tag HTML <button>

Exemplo

Um botão clicável é marcado da seguinte maneira

<button type="button> Clique aqui! <button>

Definição e Uso

A tag <button> define um botão clicavel.

Dentro do elemento <button> você pode colocar texto e tags como <i>, <b>, <strong>, <br>, <img>,etc. Isso não é possivel com um botão criado com o elemento <input>

Dica: Sempre especifique o tipo de atributo para um elemento <button>, para dizer ao navegador qual o tipo de botão é.

Atributos

Atributos Globais

A tag <button> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <button> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Utilize CSS para costumizar botões:

<!DOCTYPE html>
<html>
<head>
<style>
.button{
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
  margin: 4px 2px;
  cursor: pointer;
}

.button1 {background-color: #4CAF50;} /* Green */
.button2 {background-color: #008CBA;} /* Blue */
</style>
</head>
<body>
<button class="button button1">Green
<button class="button button2">Blue
</body>
</html>
