Tag HTML <canvas>

Exemplo

Desenhe um retângulo vermelho rapidamente e mostre-o dentro do elemento <canvas>:

<canvas id="myCanvas">
    Seu navegador não suporta a tag canvas.
</canvas>

<script>
   var canvas = document.getElementById("myCanvas");
   var ctx = canvas.getContext("2d");
   ctx.fillStyle = "#FF0000";
   ctx.fillRect(0, 0, 80, 80);
</script>

Definição e uso

A tag <canvas> é usada para desenhar gráficos, dinamicamente, por meio de scripts (geralmente JavaScript).

A tag <canvas> é transparente e é apenas um contêiner para gráficos, você deve usar um script para desenhar os gráficos.

Qualquer texto dentro do elemento <canvas> será exibido em navegadores com JavaScript desabilitado e em navegadores que não suportam <canvas>.

Dicas: Aprenda mais sobre o elemento<canvas> no tutorial canvas HTML. Para obter uma referência completa de todas as propriedades e métodos, visite nossa referência canvas HTML .

Atributos

Atributo Valor Descrição
height pixels Especifica a altura da tela. O valor padrão é 150 pixels.
width pixels Especifica a largura da tela. O valor padrão é 300 pixels.

Atributos Globais

A tag <canvas> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <canvas> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Outro exemplo de <canvas>:

<canvas id="myCanvas">
  Seu navegador não suporta a tag canvas.
</canvas>

<script>
  var c = document.getElementById("myCanvas");
  var ctx = c.getContext("2d");
  ctx.fillStyle = "red";
  ctx.fillRect(20, 20, 75, 50);
  // Ativar transparência
  ctx.globalAlpha = 0.2;
  ctx.fillStyle = "blue";
  ctx.fillRect(50, 50, 75, 50);
  ctx.fillStyle = "green";
  ctx.fillRect(80, 80, 75, 50);
</script>

CSS padrão

Exemplo

A maioria dos navegadores exibirá o elemento <canvas> com os seguintes valores padrão:

canvas {
  height: 150px;
  width: 300px;
}
