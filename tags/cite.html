Tag HTML <cite>

Exemplo

Definir o título de uma obra:

<p><cite>O Grito</cite> por Edward Munch. Pintada em 1893.</p>

Definição e uso

A tag <cite> define o título de uma obra criativa, como por exemplo, livro, poema, filme ou pintura.

O texto no elemento <cite> é normalmente renderizado em itálico

Atributos

Atributos Globais

A tag <cite> suporta os atributos globais do HTML.

Atributos de Evento

A tag <cite> suporta os atributos de evento do HTML.

CSS padrão

Exemplo

cite {
  font-style: italic;
}
