Tag HTML <del>

Exemplo

Um texto com uma parte deletada, e uma nova parte inserida:

<p>Minha cor favorita é <del>vermelho</del> <ins>azul</ins>!</p>

Definição e uso

A tag <del> define um segmento de texto que foi deletado.

Navegadores normalmente vão riscar o texto deletado com uma linha horizontal.

Atributos

Atributo Valor Descrição
cite URL Especifica uma URL para um documento que explica por qual motivo o texto foi deletado ou modificado
datetime YYYY-MM-DDThh:mm:ssTZD Especifica a data e hora em que o texto foi deletado ou modificado

Atributos

Atributos Globais

A tag <del> suporta os atributos globais do HTML.

Atributos de Evento

A tag <del> suporta os atributos de evento do HTML.

Outros exemplos

Exemplo

Usando CSS para estilizar <del> e <ins>:

<html>
<head>
<style>
  del {background-color: tomato;}
  ins {background-color: cyan;}
}
</style>
</head>
<body>

<p>Minha cor favorita é <del>vermelho</del> <ins>azul</ins>!</p>

</body>
</html>
