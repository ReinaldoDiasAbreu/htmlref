Tag HTML <details>

Exemplo

Especifica detalhes que o usuário pode abrir e fechar quando quiser:

<details>
  <summary>Epcot Center</summary>
  <p>Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.</p>
</details>

Definição e uso

A tag <details> especifica detalhes que o usuário pode abrir e fechar quando quiser.

A tag <details> é geralmente utilizada para criar um widget interativo que o usuário pode abrir e fechar. Por padrão, o widget está fechado. Quando aberto, ele se expande e exibe seu conteúdo.

Qualquer tipo de conteúdo pode ser colocado em <details>.

Dica: A tag <summary> é utilizada com <details> para especificar um cabeçalho para o conteúdo.

Atributos

Atributo Valor Descrição
open open Especifica que o conteúdo deve estar visível ao usuário

Atributos Globais

A tag <details> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <details> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Utilize CSS para decorar <details> and <summary>:

<html>
<head>
<style>
details > summary {
  padding: 4px;
  width: 200px;
  background-color: #eeeeee;
  border: none;
  box-shadow: 1px 1px 2px #bbbbbb;
  cursor: pointer;
}
details > p {
  background-color: #eeeeee;
  padding: 4px;
  margin: 0;
  box-shadow: 1px 1px 2px #bbbbbb;
}
</style>
</head>
<body>

<details>
  <summary>Epcot Center</summary>
  <p>Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.</p>
</details>

</body>
</html>
