Tag HTML <dfn>

Exemplo

Exibe a definição de um termo quando o ponteiro do mouse está sobre ele:

<p><dfn title="HyperText Markup Language">HTML</dfn> é a linguagem padrão de marcação para criação de páginas web.</p>

Definição e uso

A tag <dfn> significa “definição de elemento”, e especifica um termo que vai ser definido dentro do conteúdo.

Atributos

Atributos Globais

A tag <dfn> suporta os atributos globais do HTML.

Atributos de Evento

A tag <dfn> suporta os atributos de evento do HTML.

CSS padrão

Exemplo

dfn {
  font-style: italic;
}
