Tag HTML <figcaption>

Exemplo

Use um elemento <figure> para marcar uma foto em um documento e um elemento <figcaption> para definir uma legenda para a foto:

<figure>
  <img src="pic_trulli.jpg" alt ="Trulli" style="width:100%">
  <figcaption>Fig.1 - Truilli,puglia, Italy.</figcaption>
</figure>

Definição e Uso

A tag <figcaption> define uma legenda para um elemento <figure>.

Um elemento<figcaption> pode ser colocado como primeiro ou ultimo filho do elemento <figure>.

Atributos Globais

A tag <figcaption> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <figcaption> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Utilize CSS para costumizar <figure> e <figcaption>

<html>
<head>
<style>
figure {
  border: 1px #ccccc solid;
  padding: 4px;
  margin: auto;
}

figcaption {
  background-color: black;
  color: white;
  font-style: italic;
  padding: 2px;
  text-align: center;
}
</style>
</head>
<body>

<figure>
  <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
  <figcaption>Fig.1 - Trulli, Puglia, Italy</figcaption>
</figure>

</body>
</html>
