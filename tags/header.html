Tag HTML <header>

Exemplo

Um cabeçalho para um <artigo>:

<article>
  <header>
    <h1>Um título aqui</h1>
    <p>Postado por John Doe</p>
    <p>Algumas informações adicionais aqui</p>
  </header>
  <p>Lorem Ipsum dolor set amet....</p>
</article>

Definição e uso

O elemento <header> representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

O elemento <header> normalmente contém:

  • um ou mais elementos de título (<h1> - <h6>)
  • logotipo ou ícone
  • informação de autoria

Observação:Você pode ter vários elementos <header> em um documento HTML. No entanto, <header> não pode ser colocado em um <footer>, <address> ou outro elemento <header>.

Atributos Globais

A tag <header> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <header> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Um cabeçalho de página:

<header>
  <h1>Título da página principal aqui</h1>
  <p>Postado por John Doe</p>
</header>

CSS padrão

Exemplo

A maioria dos navegadores exibirá o elemento <header> com os seguintes valores padrão:

header {
  display: block;
}
