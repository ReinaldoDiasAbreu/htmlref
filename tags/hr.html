Tag HTML <hr>

Exemplo

Usando a tag para dividir o texto em seções:

<h1>As principais linguagens da Web</h1>

<p>HTML é a linguagem padrão de marcação para criação de páginas web.</p>

<hr>
<p>CSS é a linguagem que descreve como os elementos HTML vão ser exibidos.</p>
<hr>

<p>JavaScript é a linguagem de programação do HTML e da Web.</p>

Definição e uso

A tag <hr> define uma quebra temática em uma página.

O elemento <hr> é mais comumente exibido como uma linha horizontal que é usada para separar o conteúdo.

Atributos

Atributos Globais

A tag <hr> suporta os atributos globais do HTML.

Atributos de Evento

A tag <hr> suporta os atributos de evento do HTML.

Outros exemplos

Exemplo
