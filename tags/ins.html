Tag HTML <ins>

Exemplo

Representa um intervalo de texto que foi inserido no texto.

<p>Minha cor favorita é <del>azul</del> <ins>vermelho</ins>!</p>

Definição e uso

A tag <ins> define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido.

Dica: Utilize <del> para marcar o texto excluído.

Atributos

Atributo Valor Descrição
cite URL Especifica um URL para um documento que explica a razão pela qual o texto foi inserido / alterado.
datetime YYYY-MM-DDThh:mm:ssTZD Especifica a data e hora em que o texto foi inserido / alterado.

Atributos Globais

A tag <ins> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <ins> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Use CSS para estilizar <del> e <ins>:

<html>
 <head>
  <style>
   del {background-color: tomato;}
   ins {background-color: yellow;}
  </style>
 </head>
 <body>
  <p>My favorite color is <del>blue</del> <ins>red</ins>!</p>
 </body>
</html>

CSS padrão

Exemplo

A maioria dos navegadores exibirá o elemento <ins> com os seguintes valores padrão:

ins {
  text-decoration: underline;
}
