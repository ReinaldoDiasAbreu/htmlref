Tag HTML <mark>

Exemplo

Destaca partes do texto.

<p>Não se esqueça de comprar <mark>leite</mark> hoje.</p>

Definição e uso

A tag <mark> define o texto que deve ser marcado ou destacado.

Atributos Globais

A tag <mark> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <mark> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

<html>
 <head>
 </head>
 <body>
  <p> <mark>Texto destacado</mark>. </p>
 </body>
</html>

CSS padrão

Exemplo

A maioria dos navegadores exibirá o elemento <mark> com os seguintes valores padrão:

mark {
  background-color: yellow;
  color: black;
}
