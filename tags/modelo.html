Tag HTML <a>

Exemplo

Cria um link para o site do IFNMG

<a href="http://www.ifnmg.edu.br">Visite o site do IFNMG!</a>

Definição e uso

A tag <a> define um hiperlink, o quel é usado para ligar uma página a outra.

Atributos

Atributo Valor Descrição
download filename Especifica que o algo será descarregado quando o usuário...
href URL Especifica a URL da página para a qual o link direciona

Outros exemplos

Exemplo

Como usar uma imagem como link:

<a href="http://www.ifnmg.edu.br"> <img src="ifnmg_logo.pbg"> </a>
