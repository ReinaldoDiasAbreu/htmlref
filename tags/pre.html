Tag HTML <pre>

Exemplo

Texto pré-formatado:

<pre>
  O texto em um elemento
  pre é exibido em largura fixa fonte,
  e preserva ambos        os espaços
  e quebras de linha.
</pre>

Definição e uso

A tag <pre> define o texto pré-formatado. O texto em um elemento <pre> é exibido em uma fonte de largura fixa e preserva os espaços e as quebras de linha. O texto será exibido exatamente como escrito no código-fonte HTML.

Outros exemplos

Exemplo

Como criar um texto pré-formatado com largura fixa (com CSS)::

<div style="width:200px;overflow:auto">
  <pre>Este é um pré com largura fixa. Ele usará tanto espaço quanto especificado.</pre>
</div>

CSS padrão

Exemplo

A maioria dos navegadores exibirá o elemento <pre> com os seguintes valores padrão:

pre {
   display: block;
   font-family: monospace;
   white-space: pre;
   margin: 1em 0;
}
