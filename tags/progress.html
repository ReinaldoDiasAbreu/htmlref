Tag HTML <progress>

Exemplo

Mostra uma barra de progresso:

<label for="arquivo">Andamento do download:</label>
<progress id="arquivo" value="32" max="100">32%</progress>

Definição e uso

A tag <progress> representa o estado de conclusão de uma tarefa.

Dica: Sempre adicione uma tag <label>. É uma boa prática que ajuda na acessibilidade!

Dica: Use a tag <progress> com JavaScript para exibir o progresso de uma tarefa.

Nota: A tag <progress> não é adequada para exibir medições estáticas (como a quantidade de espaço restante em disco, por exemplo). Para representar um medidor, utilize a tag <meter>.

Atributos

Atributo Valor Descrição
max número Especifica a quantidade de trabalho total que a tarefa requer. O valor padrão é 1
value número Especifica o quanto da tarefa já foi feito

Atributos Globais

A tag <progress> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <progress> fornece suporte para os atributos de evento do HTML.
