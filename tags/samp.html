Tag HTML <samp>

Exemplo

Exibir algum texto como sendo o output de um programa:

<p>Mensagem do meu computador:</p>
<p><samp>Arquivo não encontrado.<br>Pressione F1 para continuar</samp></p>

Definição e uso

A tag <samp> é usada para definir uma amostra do output de um programa de computador.

O conteúdo é exibido na fonte padrão monospace do navegador.

Atributos

Atributos Globais

A tag <samp> suporta os atributos globais do HTML.

Atributos de Evento

A tag <samp> suporta os atributos de evento do HTML.

CSS padrão

Exemplo

samp {
  font-family: monospace;
}
