Tag HTML <select>

Exemplo

Cria uma lista drop-down

<for="carros">Escolha uma carro:</label>

<select name="carros" id="carros">
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="mercedes">Mercedes</option>
  <option value="audi">Audi</option>
</select>

Definição e uso

O elemento <select> é utilizado para criar uma lista drop-down.

O elemento <select> é mais utilizado em formulários, para coletar entradas de usuário.

O atributo name é necessário para referenciar os dados do formulário após o envio (se você omitir o atributo name, nenhum dado da lista será enviado).

O atributo id é necessário para associar a lista a um rótulo.

As tags <option> dentro do <select> definem as opções disponíveis na lista.

Dica: Sempre adicione uma tag <label>. É uma boa prática que ajuda na acessibilidade!

Atributos

Atributo Valor Descrição
autofocus autofocus Especifica que a lista deve ganhar foco automaticamente após o carregamento da página
disabled disabled Especifica que a lista deve estar desativada
form id_formulário Define a qual formulário a lista pertence
multiple multiple Especifica que o usuário pode selecionar várias opções de uma só vez
name nome Define um nome para a lista
required required Especifica que o usuário deve selecionar um valor antes de enviar o formulário
size número Define a quantidade de opções visíveis na lista

Atributos Globais

A tag <select> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <select> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Use tags <optgroup> em seu <select>:

<for="carros">Escolha uma carro:</label>

<select name="carros" id="carros">
  <optgroup label="Carros suecos">
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
  </optgroup>
  <optgroup label="Carros alemães">
    <option value="mercedes">Mercedes</option>
    <option value="audi">Audi</option>
  </optgroup>
</select>
