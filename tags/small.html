Tag HTML <small>

Exemplo

Define um texto menor

<p>Este é um texto normal.</p>
<p><small>Este é um texto menor.</small></p>

Definição e Uso

A tag <small> define um texto menor(como copyright e outros comentários laterais).

Dica: Esta tag não está obsoleta, mas é possível obter um efeito mais rico(ou o mesmo) com CSS.

Atributos Globais

A tag <small> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <small> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Usando CSS para definir um texto menor

<html>
<head>
<style>
span.small {
  font-size:smaller;
}

</style>
</head>
<body>

  <p>Este é um texto normal.</p>
  <p>Este é um texto menor.</span></p>

</body>
</html>

CSS padrão

Exemplo

small {
  font-size: smaller;
}
