Exemplo

Um elemento <span> é usado para colorir uma parte do texto

<p>My mother has <span style="color:blue">blue</span> eyes.</p>

Definição e Uso

A tag <span> é um inline container, usado para marcar parte de um texto ou parte de um documento.

A tag <span> é facilmente costumizavel com CSS ou manipulada com JavaScript usando a class ou id attribute.

A tag <span> é muito parecida com o elemento <div>, mas <div> é um elemento block-level e <span> é um elemento inline.

Atributos Globais

A tag <span> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <span> fornece suporte para os atributos de evento do HTML.
