Tag HTML <u>

Exemplo

Utilize a tag <u> para destacar error ortográficos:

<p>Este texto está <u>herrado</u>.</p>

Definição e uso

A tag <u> destaca partes do texto que distoam do resto do conteúdo, como erros ortográficos e nomes próprios chineses.

Dica: Evite utilizar a tag <u> onde ela pode ser confundida com um hiperlink!

Atributos Globais

A tag <u> fornece suporte para os atributos globais do HTML.

Atributos de Evento

A tag <u> fornece suporte para os atributos de evento do HTML.

Outros exemplos

Exemplo

Utilize CSS para decorar erros ortográficos:

<html>
<head>
<style>
.error {
  text-decoration-line: underline;
  text-decoration-style: wavy;
  text-decoration-color: red;
}
</style>
</head>
<body>

<p>Este texto está <u class="error">herrado</u>.</p>

</body>
</html>
