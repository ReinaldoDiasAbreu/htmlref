Tag HTML <var>

Exemplo

Definir algum texto como variáveis em um documento:

<p>A área do triângulo é: 1/2 x <var>b</var> x <var>h</var>, onde <var>b</var> é a base, e <var>h</var> é a altura.</p>

Definição e uso

A tag <var> é usada para definir uma variável em programação ou em uma expressão matemática.

O conteúdo é tipicamente mostrado em itálico.

Atributos

Atributos Globais

A tag <var> suporta os atributos globais do HTML.

Atributos de Evento

A tag <var> suporta os atributos de evento do HTML.
